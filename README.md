# Users dashboard table

### Used third party libraries:

- [react-hook-form](https://react-hook-form.com/) - for forms validation and handling values
- [@tanstack/react-virtual](https://tanstack.com/virtual/latest) - for virtualized tables
- [zustand](https://zustand-demo.pmnd.rs/) - for global state management

### Requirements:

- Node.js (v22.18.0)
- npm (v10.9.3)
- yarn (v1.22.22)

### Run the project:

1. Install necessary packages:

```
yarn install
```

2. Build the project:

```
yarn build
```

3. Run the project:

```
yarn start
```

4. Open http://localhost:4173/
